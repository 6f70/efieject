efieject
========

Eject!

Build
--------
1. Setup EDK2
2. add a line ```MdeModulePkg/Application/efieject/Eject.inf``` under ```[Components]``` in ```$WORKSPACE/MdeModulePkg/MdeModulePkg.dsc```
3. ```cd $WORKSPACE/MdeModulePkg/Application && git clone git@github.com:6f70/efieject.git```
4. ```cd efieject && build```
